# 支持说明

社区支持按尽力而为原则提供。本仓库支持与精选 Skill 的来源、许可证与署名、安全、可移植性、重叠、冲突、生命周期、拓扑、确定性验证和发布清单有关的问题与报告。

提交 Issue 前请：

1. 标明准确的仓库 revision，以及受影响的文件、Skill、来源、registry 记录或派生投影；
2. 说明它属于已批准发布、候选材料、历史证据、官方/运行时基线还是外部工具；
3. 提供公开安全的复现证据和所用验证命令；
4. 删除凭据、私有账号状态、个人记忆、私有配置、受限第三方内容和原始敏感日志。

恶意 Skill、泄漏材料、凭据暴露或可利用的供应链问题请使用 GitHub 私有安全公告；非敏感的治理、验证、可移植性、文档和候选来源讨论可以使用公开 Issue。

本仓库不为未批准候选的安装、账号连接、live Agent 配置或第三方服务提供支持，也不保证响应时间或服务等级。赞助不会改变这些边界。
