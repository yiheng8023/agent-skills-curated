# Native And Ecosystem Overlap Review

Source: `github:mattpocock/skills`

Revision: `6eeb81b5fcfeeb5bd531dd47ab2f9f2bbea27461`

Every historical baseline Skill was compared with ordinary Agent reasoning,
editing, instruction following, current visible runtime capabilities, and the
existing curated release. Generic writing, teaching, summarization, brevity,
perspective-shifting, scaffolding, and simple setup instructions are not a
distinct native increment. Product adapters and deprecated/in-progress
upstream content do not receive automatic approval.

Distinct procedures may remain when they define a reusable artifact contract,
non-obvious safety discipline, cross-step validation loop, or specialized
engineering method. Runtime arbitration may still prefer an equivalent native
or visible capability in a particular environment.

The per-Skill disposition and rationale are recorded in
`audits/native-overlap/2026-06-22.md` and `registry/admissions.json`.
