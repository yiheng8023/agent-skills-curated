# Native And Runtime Baseline Evidence Gap

Status: `partial` remains correct.

The dated 2026-07-15 artifact is a useful single-host metadata baseline for
four linked demand records. It binds Codex desktop on Windows, a declared model
family, unobservable reasoning detail, loader limits, repository-scoped
permissions, workspace, date, paths, hashes, and explicit non-claims. Its
historical `baselineRecordedForEveryDemandRecord=true` means every record in
that four-record linked batch, not every record in the current survey.

The current survey has eight demand records and a 62-row bounded coordinate
matrix. EL-05 through EL-08 each have an alternative review, but those reviews
are not a dimension-complete dated baseline: they do not separately record
native, official, runtime, installed, and already-authorized availability with
per-demand activation mode, reproducible probe method, and raw outcomes.
Coordinate mapping and an alternative list do not prove live availability.

The acceptance therefore remains `partial`. A future upgrade requires a
host-specific record for all eight demand records, with model, reasoning,
loader, activation, permissions, workspace, probe method, raw outcomes,
limitations, and refresh triggers. Another Agent or host requires a separate
baseline; the current evidence proves no cross-host behavior.
