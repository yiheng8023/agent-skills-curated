## Local Result

Eighteen deterministic fixtures compare native, current-chain, and exact Loopy
routes across six scenario families. All 18 contract fixtures pass.

The contract excludes Loopy as the preferred route for five families:

- native remains preferred for true one-shot analysis;
- the guarded current chain retains authority for high-impact external action;
- `diagnose` remains preferred for debugging reproduction;
- native open work remains preferred for divergent ideation;
- every route must stop when an observable gate or finite boundary is absent.

Only reversible iterative local repair remains a candidate for a controlled
Agent trial. Even there, native and the current chain remain eligible baselines;
the fixture does not prefer Loopy.

## Evidence Limit

This is contract evidence, not Agent behavior evidence. It proves that the
comparison design can avoid obvious overuse and preserve authority boundaries.
It does not prove Agent compliance, correctness improvement, lower iteration
cost, better receipts, cross-host behavior, superiority, a residual gap,
admission, adaptation, self-authoring, or Hook need.

## Proposed Controlled Trial

The next empirical gate would use fresh disposable tasks with the same local
fixture and deterministic verification surface. It would compare native,
current-chain, and exact pinned Loopy arms on iterative repair, plus a one-shot
negative control. The candidate body would be supplied only to its disposable
task, never installed or persisted.

That trial is not authorized by this protocol because it would execute an
unapproved third-party candidate body. Separate authorization is required
before creating the fresh Agent tasks.
