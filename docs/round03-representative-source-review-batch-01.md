# Round 03 Representative Source Review — Batch 01

Date: 2026-07-15

This non-executing static review covers all nine pinned representatives from
the first public discovery snapshot. License conclusions bind concrete files
and Git blob identities rather than GitHub labels alone.

The review separates source roles before comparison:

- two official repositories remain external overlap baselines;
- one index remains a child-source radar surface;
- `handoff-skill` and `planning-with-files` remain Skill-system comparisons;
- Tank, qvr, MagicSkills, and agnix are non-Skill tooling or architecture
  alternatives, not Skill admission candidates.

`handoff-skill` is not an exact duplicate of the installed `.cc-switch`
`handoff`: the pinned bodies are 6076 and 847 bytes with different SHA-256
values. It is a candidate-with-limits for content and alternative comparison,
not for installation or admission.

`planning-with-files` has 235 executable Hook/script files and overlaps
planning, continuation, closure, installed handoff, and the candidate Hook
policy. It remains a heavy-system reference. Tank, qvr, MagicSkills, and agnix
have 784, 434, 42, and 205 executable source/test files respectively, so each
requires a separate tool-level security and dependency assessment if later
selected.

No source is approved or executable. The next gate is demand-level alternative
comparison, including native and already-installed paths, project standards,
and human authority before any residual-gap or Hook decision.
