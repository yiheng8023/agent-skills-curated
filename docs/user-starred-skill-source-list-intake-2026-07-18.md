# User-Starred Skill Source List Intake

Date: 2026-07-18

The user-provided public GitHub list is now a supplemental discovery signal,
not a whitelist or a discovery boundary. Its 13 visible repositories overlap
the broad public discovery by eight and add five previously unseen sources.

A star raises review priority but proves neither quality nor safety. Every
source still requires immutable revision pinning and separate treatment of
indexes, tooling, suites, Skill bodies, and Hooks, followed by license,
provenance, security, portability, overlap, redundancy, and validation review.

Discovery continues outside this list through public search, official
baselines, trusted indexes, and shortfall-directed queries. No star, list
membership, install, execution, Hook enablement, adaptation, or admission was
changed by this intake.
